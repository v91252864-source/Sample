# Sample

## Usage

- Open the sample site locally by opening `index.html` or `login.html` in a browser.
- Or serve the folder with a simple HTTP server:

```bash
python3 -m http.server 8000

# then open http://localhost:8000/ in your browser
```

This repository contains:

- `index.html` — a simple sample page
- `login.html` — a responsive login form (assets in `assets/`)
